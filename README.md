
# SQL Data Warehouse Development

A hands-on SQL Data Warehouse project built using SQL Server to implement a modern data warehouse following Medallion Architecture.

## Project Overview

This project demonstrates the implementation of an end-to-end data warehouse by integrating ERP and CRM datasets into a centralized analytical database. It covers data ingestion, transformation, dimensional modeling, and business-ready reporting structures.

## Architecture

The project follows the Medallion Architecture:

- Bronze Layer: Raw data ingestion from source systems
- Silver Layer: Data cleansing, standardization, and transformation
- Gold Layer: Business-ready dimensional model using Star Schema

## Key Features

- SQL Server Data Warehouse
- ETL Pipelines
- Bronze, Silver, and Gold Layers
- Star Schema Data Modeling
- Fact and Dimension Tables
- Data Quality Validation
- Business-ready Analytical Model

## Tech Stack

- SQL Server
- SQL Server Management Studio (SSMS)
